# 🏥 Healthcare Patient Analytics — Understanding Readmissions  
### 📊 Power BI + Python + SQL Project  

> 🚀 Designed & Developed by me

---

## 📌 Project Overview  

This project analyzes **hospital patient readmissions** to identify high-risk factors and improve healthcare decision-making. It simulates a real-world analytics workflow and delivers insights through an interactive **Power BI dashboard**.

---

## 🎯 Objectives  

- 🔍 Identify patients at high risk of readmission  
- 📈 Analyze demographic and clinical patterns  
- 🏥 Optimize hospital resource utilization  
- 📊 Build an interactive dashboard for decision-making  

---

## 🧠 Key Insights  

- ⚠️ **Heart Failure & Kidney Disease** have highest readmission rates (~35%)  
- 👴 Patients aged **70+** show 2.5× higher readmission probability  
- ⚖️ Gender has minimal impact on readmission  
- 💰 Treatment costs range between **$5,000 – $100,000**  

---

## 🏗️ Project Workflow  

### 🔹 Step 1: Data Generation  
- Created synthetic dataset (1,000 patients, 12,000 admissions)  
- Included demographics, diagnoses, procedures, charges  
- Applied realistic readmission logic  

---

### 🔹 Step 2: Data Cleaning  
- No missing values  
- Optimized data types  
- Reduced memory usage  

---

### 🔹 Step 3: Exploratory Data Analysis (EDA)  

- Age distribution analysis  
- Diagnosis-based readmission trends  
- Pattern identification across features  

---

### 🔹 Step 4: Dashboard Development  

- KPIs: Admissions, Charges, LOS, Readmission Rate  
- Filters: Year, Insurance, Gender, Age Group  
- Visuals: Diagnosis trends, monthly admissions  

---

## 📊 Dashboard Features  

- 📌 Interactive filters  
- 📉 Readmission analysis by diagnosis  
- 📅 Monthly admission trends  
- 💰 Cost and revenue insights  
- 🏥 Operational metrics  

---

## 🚀 Business Recommendations  

- 🎯 Focus on high-risk diagnoses  
- 👴 Improve elderly patient care (70+)  
- 📞 Enhance post-discharge follow-ups  
- 📊 Use dashboard for real-time monitoring
